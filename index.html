Política de Privacidade

Última atualização: 10/12/2024

Seu direito à privacidade é nossa prioridade. Nosso aplicativo [Nome do Aplicativo] foi desenvolvido para oferecer uma experiência segura e respeitosa, sem coletar, armazenar ou compartilhar informações pessoais dos usuários.

Coleta de Informações

Nosso aplicativo não coleta nenhuma informação pessoal, como nome, endereço de e-mail, localização ou qualquer dado que possa identificá-lo.

Permissões do Dispositivo

O aplicativo pode solicitar permissões limitadas, como acesso à internet, apenas para funcionamento técnico ou exibição de conteúdo. Nenhuma informação coletada por essas permissões é armazenada ou compartilhada.

Uso de Dados

  • Dados Coletados: Não coletamos dados pessoais.
  • Dados Anônimos: Dados de uso anônimos podem ser processados (como estatísticas de uso) para melhorar o aplicativo. Esses dados não contêm informações que possam identificar você.

Compartilhamento de Informações

Não compartilhamos informações com terceiros, pois nenhuma informação pessoal é coletada.

Alterações na Política de Privacidade

Podemos atualizar esta política ocasionalmente. Recomendamos que os usuários revisem esta página regularmente para se manterem informados.
